# Alfie Golf Trainings — Concierge Test

This folder contains personalised practice plans for all golfers in the concierge test cohorts.

## Folder structure

Each golfer has their own numbered folder. Files are named by golfer and week — no subfolders.

```
alfie-golf-trainings/
├── 01-adi-dobson/
│   ├── adi-dobson-week1-plan.html
│   ├── adi-dobson-week1-plan.pdf
│   ├── adi-dobson-week2-plan.html
│   └── adi-dobson-week2-plan.pdf
├── 02-[firstname-lastname]/
│   └── ...
```

## File naming convention

- HTML: `[firstname-lastname]-week[n]-plan.html` — interactive, fillable, send button wired to alfie.golf.trainings@gmail.com
- PDF:  `[firstname-lastname]-week[n]-plan.pdf`  — static, attach to email alongside HTML
